Shooting Game

  1. Boxes will start falling towards the ship.
  2. The game will end when the time is over 2 minute or boxes reach the ship.
  3. Moving the ship using keyboard arrows and shooting using the space button.
  4. Red boxes will destroy surrounding boxes with one bullet and have a score of +15.
  5. Simple destruction of one box will give the user 2 as a score.
  6. The user will win before time if they remove 300 boxes.
